# 🌌 Wonders of the Universe

A modern, interactive, frontend-only educational web application that lets users explore the beauty of the universe. Discover the Solar System, planets, moons, stars, galaxies, black holes, astronauts, and historic space missions through an immersive NASA-inspired user interface.

## 🚀 Live Demo

🌐 **https://wonders-of-the-universe.vercel.app**

> Replace the URL above with your actual Vercel deployment if it is different.

---

## 📖 Project Overview

**Wonders of the Universe** is a premium educational website developed using **HTML, CSS, and JavaScript**. It provides an engaging way to learn about space with interactive animations, responsive layouts, modern UI, and detailed information about celestial objects.
